# bhargava-morampalli/rnamodbench: Documentation

The bhargava-morampalli/rnamodbench documentation is split into the following pages:

- [Operator Scripts Runbook](operator_scripts_runbook.md)
  - Canonical operations guide for `generate_samplesheets.sh`, `run_all_coverages.sh`,
    `bin/downstream_analysis/collate_runs.py`, and `bin/downstream_analysis/run_analysis.py`.
- [Usage](usage.md)
  - An overview of how the pipeline works, how to run it and a description of all of the different command-line flags.
- [Output](output.md)
  - An overview of the different results produced by the pipeline and how to interpret them.

You can find a lot more documentation about installing, configuring and running rnamodbench pipelines on the website: [https://github.com/bhargava-morampalli/rnamodbench](https://github.com/bhargava-morampalli/rnamodbench)
